春雏集-看雨

发表于 2017-03-30 | 分类于 Poem | | 阅读次数

旧枝抽新芽，忍把黄叶捡，
水莹莹，灯晃晃，
愁云压黑一片霓虹。

不怕三月连绵雨，恐伤我四月花。
只叹稠雨空逝水，不言叶落红泥。

小池塘积水潭，荷叶连连时复时。
最是一年春好处，等闲偷来看雨。

                ——lvc.17.3.30.晚
